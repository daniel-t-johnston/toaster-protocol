# toaster-protocol
CSC 242 Group Project Repo

## Solution Projects and Important Files

### Cipher Assignment
- #### Cipher: Compiled to create Cipher.exe
  - ##### Source.cpp: Contains main(), which is used to call functions from the Cipher.Core static library using Cipher.h
- #### Cipher.Core: Cipher static library
  - ##### Cipher.cpp: Contains functions for the Cipher program
  - ##### Cipher.h: Header file for Cipher.Core. All exposed functions need to be included.
- #### Cipher.Test: Testing for Cipher
  - ##### Cipher.Test.cpp: Tests for Cipher.Core functions

### SpellCheck Assignment
- #### SpellCheck: Compiled to create SpellCheck.exe
  - ##### Source.cpp: Contains main(), which is used to call functions from the SpellCheck.Core static library using SpellCheck.h
  - ##### words.txt: Dictionary for SpellCheck
- #### SpellCheck.Core: SpellCheck static library
  - ##### SpellCheck.cpp: Contains functions for the SpellCheck program
  - ##### SpellCheck.h: Header file for SpellCheck.Core. All exposed functions need to be included.
- #### SpellCheck.Test: Testing for SpellCheck
  - ##### SpellCheck.Test.cpp: Tests for SpellCheck.Core functions
